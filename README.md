# 🔐 Password Generator

A secure and customizable **Password Generator** built with **HTML**, **CSS**, and **JavaScript** that helps you to generate strong, random passwords instantly based on your preferences.

---



## ✨ Features

- 🎚️ Adjustable password length using a slider
- 🔢 Option to include numbers
- 🔣 Option to include special symbols
- 🔠 Option to include uppercase letters
- 📋 Copy password to clipboard with one click

---

## 🎯 Project Purpose

This project was created to strengthen the following concepts:

- 🔄 DOM Manipulation and dynamic content rendering
- 🖱️ Event Handling for checkboxes, sliders and buttons
- 🎲 Generating secure random characters using JavaScript
- 📱 Building a fully responsive and interactive UI

---

## 🛠️ Technologies Used

- **HTML5** – Structure and markup
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6+)** – Password generation logic 

---




---

## 📌 How To Use

1. Move the **slider** to set your desired password length.
2. Check the boxes to include **numbers**, **symbols**, or **uppercase letters**.
3. Click the **Generate** button.
4. Click the **Copy** icon to copy the password to your clipboard.
5. Use your strong password anywhere! 🔒
